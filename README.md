# Server Functionalities

### So Your Presentation Is About Offline Video Meeting Real Time Web Application Which Uses The Following Packages And Library : - 
<p>1. HTML CSS Javascript Bootstrap
<br>2. Nodemon ( To Automate Restart Of Server Whenever Changes Are Made )
<br>3. ExpressJS ( To Establish Server Connection )
<br>4. WebRTC ( For Default Communication Establishment )
<br>5. Arora SDK ( Additional Packages For WebRTC )
<br>6. Mongoose (For Creation Of Mongo Db Schema)
</p>
