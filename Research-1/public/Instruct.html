Instruction To Connect To The Meeting Room

  • Setup The Browser Environment To Access The Meeting Web Application. To Setup The Environment Copy And Open This URl To Authorize - chrome://flags/#unsafely-treat-insecure-origin-as-secure

  • Enter The Following URL's In The Secure URL Box Of The Browser As Follows http://(Server IP Provided):(Port Number).

  • The Port Numbers Available Are - 3000 And 5000 And 3000 Is Set As Default. The URL's Should Be Comma Seperated For Correct Implementation.

  • Now Set The Secure Box Details To Enable And Relaunch The Browser. You Are Good To Go.

  • Enter The Link In The Browser Search Box - http://(Server IP Provided):3000

  • Now Will Get Connected To The Sign Up Page. Congrats.

  • Allow For The Required Permission When Asked For And Continue By Signning To Auto Talk.
